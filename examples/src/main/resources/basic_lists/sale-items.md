# sale-items.taq

$ java -jar taq.jar sale-items

```

mug USD5.66
cap USD15.00

```

### Description

sale-items.taq demonstrates growing a list by appending items to it. The "format_stock" 
query produces a text list with each item containing information originating from a
"stock" axiom list
